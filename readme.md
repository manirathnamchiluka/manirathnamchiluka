Chiluka Manirathnam is a third-year **B.Tech Cybersecurity** undergraduate at **CMR College of Engineering & Technology**, Hyderabad, India (Expected Graduation: 2027 | CGPA: 7.5/10). Holding the **Certified Junior Web Application Penetration Tester (JWPT)** and **APISEC Certified Practitioner (ACP)** credentials, Manirathnam brings hands-on, lab-driven experience in offensive security with a primary focus on web application exploitation, API attack surfaces, and wireless security research.

With a systematic approach to vulnerability identification, Manirathnam has developed proficiency in SQL Injection, Cross-Site Scripting (XSS), CSRF, Authentication Bypass, and API security testing — responsibly practiced across controlled lab environments, TryHackMe platforms, and independent research setups. Operating natively on **Kali Linux** and working with hardware tools including **HackRF One** and **Flipper Zero**, the research scope extends into RF signal analysis and Medical IoT security — demonstrating a rare blend of application-layer and hardware-layer offensive security awareness.

Security research projects cover diverse attack domains: from Rolling Code interception simulations and ECG signal manipulation on healthcare devices, to building Python-based malware detection tooling for reverse shell identification. All research is conducted in ethical, controlled environments with a clear emphasis on responsible disclosure and countermeasure documentation.
